# Ghost Protocol / Siro-main

Ghost Protocol is an autonomous AI workforce for the modern job search. It comprises six specialized agents that operate in concert to discover opportunities, analyze your profile, tailor your resume, craft cover letters, and execute applications.

## Getting Started

Please see the following documents for more details:
- [Context.md](./Context.md): The problem statement and system goals.
- [Architecture.md](./Architecture.md): How the system is designed.
- [Setup.md](./Setup.md): Instructions on how to set up the backend and frontend.

## Key Features
- **Zero-Touch Applications**: From discovery to submission, fully automated.
- **ATS Invisibility**: Engineered to pass applicant tracking systems.
- **24/7 Active Scanning**: Continuous job market monitoring.
- **Semantic Intelligence**: Advanced NLP matching logic.
